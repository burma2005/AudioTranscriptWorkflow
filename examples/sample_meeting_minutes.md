## 參、事件與問題檢討 (本月共 5 件)
1. **測試環境資料庫連線中斷 (03/04)**
   - 狀況：自動化測試無法執行，影響 2 個開發小組的進度約半天。
   - 根因：資料庫連線池上限設定過低，尖峰時段資源耗盡。
   - 處置：上調連線池上限並加入監控告警，已恢復正常。

2. **第三方金流 SDK 版本衝突 (03/08)**
   - 狀況：升版後出現 API 簽章驗證失敗，影響沙箱環境測試。
   - 處置：鎖定舊版 SDK 並聯繫原廠確認，預計 03/20 出新版修正。

3. **生產環境設定檔未同步問題 (03/11)**
   - 狀況：部署後發現部分環境變數未正確寫入，造成功能異常約 20 分鐘。
   - 處置：已加入部署前自動比對設定檔的驗證步驟，後續更新流程中強制觸發。
